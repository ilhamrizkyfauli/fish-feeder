# 🐟 Web Dashboard: Automatic Fish Feeder (IoT)

Aplikasi berbasis *web* yang dibangun menggunakan **Laravel** untuk memonitoring dan mengontrol sistem *Automatic Fish Feeder* (Pemberi Pakan Ikan Otomatis) berbasis *Internet of Things* (IoT). 

Sistem *web* ini terintegrasi dengan perangkat keras mikrokontroler (ESP32) dan aktuator (Motor Servo) untuk memudahkan pengguna dalam mengatur jadwal pemberian pakan ikan secara presisi dari jarak jauh.

## ✨ Fitur Utama
* **Dashboard Monitoring:** Menampilkan status perangkat dan riwayat pemberian pakan secara *real-time*.
* **Penjadwalan Otomatis:** Antarmuka untuk mengatur waktu dan porsi pakan yang akan dieksekusi oleh perangkat keras.
* **Manajemen Data:** Mengelola basis data operasional alat menggunakan antarmuka yang ramah pengguna.

## 🛠️ Teknologi yang Digunakan
* **Framework Backend:** Laravel (PHP)
* **Frontend:** Blade Templating Engine, HTML, CSS
* **Database:** MySQL
* **Integrasi Perangkat Keras:** ESP32 & Motor Servo
